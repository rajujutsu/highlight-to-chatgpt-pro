Workflow Stacks: 3 Buttons That Cover 80%

The best setup isn’t 30 templates. It’s 3 templates you actually use. Here’s the “daily driver” stack that works for most people.

The Daily Driver

1) Summarize (3 layers) · 2) Extract action items · 3) Rewrite (tone)

Stack #1: Reading

Summarize (3 layers)

Extract action items

Turn into notes

Stack #2: Writing

Rewrite (tone controls)

Improve clarity

Generate outline

Copy-ready: Stack #1 prompts

BUTTON 1 — SUMMARIZE (3 LAYERS) ROLE: You are an expert summarizer. TASK: Summarize the highlighted text using 3 layers. OUTPUT: 1) TL;DR (1–2 sentences) 2) Key takeaways (3–6 bullets) 3) Action items (checkbox list; if none, say "No clear action items.") HIGHLIGHT: {HIGHLIGHT} BUTTON 2 — ACTION ITEMS ROLE: You are an operations assistant. TASK: Extract action items from the highlighted text. CONSTRAINTS: If none exist, say "No clear action items." OUTPUT: Checkbox list + owners/deadlines if mentioned HIGHLIGHT: {HIGHLIGHT} BUTTON 3 — NOTES ROLE: You are a research assistant. TASK: Convert the highlight into a reusable note. OUTPUT: Title, Summary, Key points, Terms, Quotes, Open questions HIGHLIGHT: {HIGHLIGHT}

How to pick your 3

Look at your last 7 days online. What did you do most? Reading? Writing? Messaging? Build a stack around that.
