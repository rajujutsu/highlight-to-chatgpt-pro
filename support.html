🔑 Key recovery

Search your email for Gumroad or “Courier Web Pro”. Check Spam/Promotions.

💻 Device limit

Each license allows 2 devices. Reinstalls/new Chrome profiles can consume slots.

🔄 Reset devices

Clears your device list. Allowed once every 30 days.

🔑 Lost your license key?

  • Search your inbox for Gumroad
  • Search for Courier Web Pro (or your old product name)
  • Check Spam and Promotions
  • Make sure you’re checking the email used at checkout

🔄 Reset devices (30-day cooldown)

  • Open extension Options
  • Go to Pro / License
  • Click Reset devices
  • If you’re on cooldown, you’ll see the next reset time

🧰 `409 (Conflict)` from verify-license

  • Usually means device_limit (3rd device attempt)
  • Fix: Reset devices → Activate again
  • If you just reinstalled Chrome: you may have consumed a new slot

🧰 `401 Invalid JWT` / `invalid_token`

  • Common causes: expired token, device storage cleared, or token mismatch
  • Fix: Options → Reset Pro → Activate again
  • If it keeps happening, open a ticket and include the exact error text

🧾 When opening an issue, include:

  • Expected vs actual behavior
  • Browser + OS
  • Exact error message (copy/paste)
  • Optional screenshot
  • Optional redacted key: ABCD-****-****-WXYZ
