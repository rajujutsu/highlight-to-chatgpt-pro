🔑 Key recovery

Search your email for Gumroad or the product name. Check Spam/Promotions.

💻 Device limit

Each license allows 2 devices. Reinstalls/new profiles can consume slots.

🔄 Reset devices

Clears your device list. Allowed once every 30 days.

🔑 Lost your license key?

  • Search your inbox for Gumroad
  • Search for Highlight → Ask ChatGPT Pro
  • Check Spam and Promotions
  • Make sure you’re checking the email used at checkout

🔄 Reset devices (30-day cooldown)

  • Open extension Options
  • Go to Pro / License
  • Click Reset devices
  • If you’re on cooldown, you’ll see the next reset time

🧰 `409 (Conflict)` from verify-license

  • Usually means device_limit (3rd device attempt)
  • Fix: Reset devices → Activate again

🧰 `401 Invalid JWT` / `invalid_token`

  • Common causes: server secret changed, expired token, missing Bearer token
  • Fix: Options → Reset Pro → Activate again
