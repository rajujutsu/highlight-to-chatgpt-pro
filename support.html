Quick start

  • Highlight text on any page.
  • Right-click → choose a Courier Web action.
  • Or use your keyboard shortcut (recommended).
Tip: If a page blocks selection, try selecting a smaller portion, or use the capture/region tool (if enabled).

Set / change keyboard shortcuts

In Chrome, open chrome://extensions/shortcuts and find Courier Web. Assign a shortcut for your favorite action.

  • Shortcuts only run when Chrome allows it (some pages may restrict shortcuts).
  • If your shortcut conflicts, choose a different combo.

Pro activation

If you purchased Pro, enter your license key in the extension’s Pro/Unlock screen. After activation, Pro features (templates, favorites, advanced workflows) unlock on this device.

  • Make sure you copied the entire key (no extra spaces).
  • Use the same key you received from Gumroad.

Device limits & “Limit reached”

Pro licenses support a limited number of device activations to prevent sharing. If you reinstall Chrome, switch computers, or load the extension in multiple Chrome profiles, you may hit the activation limit.

Common cause: uninstalling/reloading the extension during development can look like a “new device” activation. If you’re testing locally a lot, you may reach the limit faster than a normal customer would.
How do I fix “Pro: limit reached”?

First, confirm you’re using the same Chrome profile and not a second profile/browser. Then try these steps:

  • Disable and re-enable the extension (keeps local storage).
  • Avoid uninstalling/reinstalling repeatedly during testing.
  • If you recently changed computers or reset your browser, request an activation reset (see below).

Activation reset: Email support with your license key and a short note (e.g., “new laptop” / “Chrome reset”). We’ll reset your activations when appropriate.

Templates/favorites are missing or reset

Templates and favorites are stored locally in your browser. They can be lost if:

  • You uninstall the extension (Chrome may remove its local storage).
  • You switch Chrome profiles.
  • Your browser clears site/extension data.

Tip: Keep one main Chrome profile for Pro, and avoid frequent uninstall/reinstall cycles.

Right-click menu doesn’t show Courier Web

Try:

  • Open chrome://extensions and confirm the extension is enabled.
  • Refresh the webpage and try again.
  • Some special pages (Chrome Web Store, internal pages) do not allow extensions to run.
Shortcuts don’t work on some sites

Some sites and special pages restrict extension shortcuts. Test on a normal webpage first. You can always use right-click as a fallback.

Image analysis / capture region not working

If a site blocks capture or the image is protected, try:

  • Use the region capture tool (if enabled) instead of right-click image.
  • Zoom in and capture only the relevant area.
  • Refresh the page and retry.
