Courier Web does not sell your data, does not run ads, and does not track your browsing. The extension only processes content when you trigger an action.

What the extension does

Courier Web lets you capture content from any webpage — selected text, images, screen regions, or YouTube transcripts — and send it to an AI tool, note-taking app, or social platform of your choice. It also includes a side panel assistant, a page monitor, and optional direct AI API integrations.

Everything is on-demand. The extension reacts to your actions; it does not silently monitor or collect browsing data in the background.

What data we collect

We do not collect personal information. Specifically:

  • No analytics or usage trackers embedded in the extension.
  • No background browsing history collection.
  • No sale or transfer of your data to third parties.
  • No advertising of any kind.
The only exception is Pro license verification — see the Pro section below.

What's stored on your device

Courier Web stores the following locally in your browser using chrome.storage.local:

  • Settings — shortcuts, UI preferences, enabled destinations, and feature flags.
  • Templates & favorites — your custom prompt templates, saved locally.
  • Action history — a local log of your recent sends and captures (up to 200 entries on free, 1,000 on Pro). Stored locally only, never transmitted.
  • Page monitors — URLs and watched text snippets you've set up for change alerts. Stored locally only.
  • Pro activation status — whether your license has been verified, stored locally.
All of this stays on your device. It is not synced to any Courier Web server. If your browser syncs extension storage via your own browser account (e.g. Chrome Sync), that is governed by your browser's own privacy policy.

When data leaves your device

Data may leave your device only in these specific, user-triggered situations:

  • Opening an AI tool or destination: When you trigger an action, your selected text or captured content is passed to the destination you chose (e.g. ChatGPT, Claude, Notion). What those services do with it is governed by their own privacy policies.
  • Translation: If you use the translate feature, your selected text is sent to Google Translate's API. No identifying information is included.
  • Page monitor checks: The monitor feature periodically fetches the URL you are watching to check for text changes. No data about the check is sent to Courier Web's servers.
  • Pro license verification: If you activate a Pro license, a verification request is sent to our server. See the Pro section below.

Direct AI API calls

Courier Web includes an optional side panel assistant and briefing features that can make direct API calls to AI providers using API keys you supply yourself. These providers include OpenAI, Anthropic, Groq, OpenRouter, and others.

  • Your API keys are stored locally on your device only.
  • API requests are sent directly from your browser to the provider — they do not pass through any Courier Web server.
  • What each provider does with the content you send is governed by their own privacy policies.
Courier Web never sees your API keys or the content of your AI conversations.

Pro license verification

If you activate a Pro license, the extension sends a verification request to our server. This request includes:

  • license_key — the key you entered.
  • A timestamp.
  • A minimal device identifier used only to enforce activation limits across devices.

We do not collect your name, email, browsing history, passwords, or payment details inside the extension. Payment is handled entirely by the third-party storefront (e.g. Gumroad) you purchased through.

Permissions explained

Courier Web requests the following Chrome permissions and uses them only as described:

  • activeTab — temporarily accesses the active page only after you trigger an action, to read your selection or prepare content for sending.
  • clipboardRead — reads your clipboard only when you explicitly trigger a paste action inside the side panel composer.
  • clipboardWrite — copies your selected text or a prepared prompt to the clipboard when you trigger a copy action.
  • contextMenus — adds the Courier right-click menu that appears when you highlight text or right-click an image.
  • scripting — injects small scripts into the active tab after a user action to retrieve selected content or prepare a destination page for pasting.
  • storage — saves your settings, templates, history, and Pro activation status locally on your device.
  • tabs — opens or focuses a destination tab when you trigger an action.
  • sidePanel — displays the Courier assistant in Chrome's built-in side panel, opened only by your direct action.
  • offscreen — creates a hidden document to perform clipboard tasks that require a document context, only after a user action.
  • pageCapture — captures the visible area of the active tab when you trigger a region or screenshot capture action.
  • notifications — shows a system notification when a page monitor you set up detects a change to the text it is watching.
  • alarms — schedules periodic checks for the page monitor feature and housekeeping tasks such as clearing temporary resources.
  • idle — detects when you are inactive so the extension can clean up temporary resources like offscreen documents.

Third-party services

When you use Courier Web to send content to a destination, that destination's own privacy policy applies. This includes services such as ChatGPT, Claude, Gemini, Notion, Slack, Google Docs, and others. Courier Web does not control what those services do with content you send to them.

The translation feature uses the Google Translate API. The AI API features use providers you configure yourself (OpenAI, Anthropic, Groq, etc.). Pro license verification uses our own server hosted on Supabase.

Data deletion

You can remove all locally stored data at any time by:

  • Uninstalling the extension (clears all local storage).
  • Using the Reset / Clear data option in the extension's Settings page.
  • Clearing the extension's storage manually in your browser's developer tools.

For Pro license data stored on our server (device activation records), contact us at the email below and we will delete it.
