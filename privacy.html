Privacy Policy — Courier Web

Last updated: February 1, 2026

Courier Web (“the extension”) helps you highlight text, analyze images, and capture screen regions, then send that content to your AI tools when you choose to. Courier Web is designed to be privacy-first: it does not track you, it does not scrape pages in the background, and it only acts when you trigger it.

Simple summary: We don’t run analytics. We don’t sell data. We don’t collect browsing history. Content is only sent to a third party (like chatgpt.com) when you trigger an action.

What data Courier Web processes

1) Content you choose to send (user-initiated)

Examples:

  • Highlighted text you select
  • Images you right-click to analyze (e.g., charts, diagrams)
  • Captured screen regions (if you use region capture)
  • Optional context like page URL and title (when enabled / used in templates)

This content is processed only when you trigger the extension (context menu, keyboard shortcut, or buttons).

2) Local storage (saved on your device)

Stored locally to power features like history and templates:

  • Template text you create (including variables like {{text}}, {{url}}, {{title}})
  • Favorites / preferences
  • Optional local history entries (selected content + URL/title + timestamps)
  • UI/settings (menus, themes, shortcuts, toggles)

This data stays inside your browser using Chrome extension storage APIs. It is not sent to the developer.

3) Pro license verification (optional)

If you unlock Pro, you may enter a license key. The key is used only to verify eligibility for Pro features.

  • Sent: the license key (and minimal verification metadata)
  • Not sent: your browsing history, highlighted text, images, or captured regions

Where data may be sent

Courier Web only shares data in these cases:

  • Your AI tool / website (e.g., chatgpt.com) — only when you trigger an action to send text/images. That site’s privacy policy applies.
  • License verification service — only when you activate Pro (license key verification only).

What we do NOT collect

  • No ads, no trackers, no analytics SDKs
  • No keystroke logging
  • No background scraping / automated browsing
  • No sale of personal data
  • No passwords, payment info, health data, or precise location data

Remote code & tracking

  • The extension does not execute remote code.
  • No tracking scripts, analytics tools, or advertising SDKs are used.

Your control

  • All processing is initiated by you.
  • You can remove local data by uninstalling the extension or clearing extension storage.

Changes to this policy

This policy may be updated if features change or requirements change. Updates will be posted on this page with a new “Last updated” date.
