Courier Web

Complete User Guide  ·  March 2026

Courier Web is a browser extension that brings an AI prompt toolbar to every webpage you visit. Select text or images, choose a prompt, and send instantly to 19+ AI and writing destinations — all without leaving your current page.

✦
Highlight & Prompt Select text or images on any page, choose a prompt, send to 19+ destinations.
◈
Side Panel Chat A full AI assistant that lives alongside any webpage in your browser.
▶
YouTube AI Fetch transcripts and run AI prompts on any YouTube video.

1Installation & Setup

Courier Web works with Chrome, Edge, Brave, Arc, and any other Chromium-based browser. Firefox and Safari support is coming soon.

1.1 Installing from the Store

1
Open the extension storeIn Chrome: Chrome Web Store. In Edge: Microsoft Edge Add-ons. In Brave or Arc: use the Chrome Web Store.
2
Search for Courier WebType "Courier Web" in the search bar and press Enter.
3
Click Add to BrowserClick Add to Chrome (or Edge / Brave) and confirm the permission prompt.
4
Pin the extensionClick the puzzle piece icon in your toolbar, find Courier Web, and click the pin icon 📌 to keep it visible.
5
Open the side panelClick the Courier Web icon. The side panel opens on the right side of your browser window.
💡
TipPinning the extension is important — it gives you one-click access to the side panel and keeps the icon always visible.

2Highlight & Prompt

Highlight & Prompt is Courier Web's core tool. Select any text and a small toolbar appears near your selection — giving you instant one-click access to AI actions and your chosen destinations.

2.1 How It Works

1
Select textClick and drag to highlight any text on a webpage. The toolbar appears automatically.
2
Click a Quick ButtonUp to three buttons appear. Click any to send your text with that prompt to your destination.
3
Or right-click for moreRight-click your selection to see the full menu: built-in tools, custom templates, and capture options.
4
Your destination opensThe prompt and selected text are sent to your chosen destination — e.g., a new ChatGPT tab opens with the prompt pre-filled.
💡
By default the toolbar only appears when you've selected at least 12 characters. Change this in Settings → Minimum character length — set it to 0 to always show.

2.2 Quick Buttons

The toolbar shows up to three Quick Buttons — configurable one-click actions. Each can be set to Weave (default destination) or a custom template. Free users get one button; Pro users get all three.

Quick Buttons are fully customizable with themes, colors, and shapes. There are over 50 built-in themes including Core (Default, Neon, Minimal, Ocean, Stellar, Aurora), Seasonal, Holiday, and fun themes like Star Wars, Pokémon, and Hologram.

You can change your default destination in Settings, or set per-site rules to route different websites to different destinations.

💡
For AI chat destinations, Courier Web opens a new tab and injects your text directly into the chat input. For social destinations, it opens the site and pastes into the appropriate compose box.

2.5 Capturing Images

  • Capture Image — right-click any image on a page and choose Capture image. The image is copied and sent to your destination.
  • Capture Region Pro — use Cmd/Ctrl + Shift + U, then click and drag to draw a box over any area of the screen.
⭐
Pro featureScreen region capture is a Pro feature. Free users can capture images already on the page.

2.6 Custom Templates

Pro users can create reusable prompt templates that appear in the right-click menu and can be assigned to Quick Buttons. Templates support placeholder variables:

{{text}}Your highlighted text. If omitted, it's appended automatically. {{url}}The URL of the current page. {{title}}The title of the current page. {{audience}}A reusable variable you set once in Settings (e.g. "non-technical managers"). {{tone}}A reusable variable you set once in Settings (e.g. "friendly and concise"). {{goal}}A reusable variable you set once in Settings (e.g. "persuade to sign up").
💡
Example template
Name: Tweet this  |  Prompt: Rewrite the following for Twitter in under 280 characters, punchy and direct: {{text}}

To create a template: open Settings → scroll to Custom Templates → enter a name and prompt → click Add template. Templates sync across browser profiles via Chrome sync storage.

3Side Panel Assistant

The side panel is a full AI chat interface that opens alongside any webpage. It persists as you navigate between pages, so you can reference what you're reading while you chat.

3.1 Opening the Side Panel

  • Click the Courier Web icon in your browser toolbar.
  • Or use the keyboard shortcut: Cmd/Ctrl + Shift + K.
  • The panel slides in on the right side of your window.
  • Resize it by dragging the left edge.
  • The panel stays open as you navigate between tabs and pages.

3.2 Choosing a Provider & Model

The side panel supports multiple AI providers. Switch between them from the dropdown at the top of the panel.

3.3 Setting Up a Local Model (Ollama)

Run AI models locally on your computer with no API key and no internet connection required.

1
Install OllamaDownload and install from ollama.com. On Mac it starts automatically in the background.
2
Pull a modelIn your terminal run: ollama pull llama3.2 (downloads ~2 GB, one-time)
3
Verify it's runningIn your terminal: curl http://127.0.0.1:11434 — you should see a response.
4
Open the side panelClick the Courier Web icon in Chrome.
5
Add the modelIn Settings, enter your model details (see table below) and click Save.
6
Allow localhostChrome will ask to allow access to 127.0.0.1 — click Allow.
7
Start chattingSelect your model from the dropdown and type a message.

Display nameLlama 3.2 (or anything you like)

"address already in use" error

3.4 Auto-Use Selection

When you have text highlighted on the current page, the side panel can automatically include it in your next message. Look for the Auto-use selection toggle in the side panel settings.

3.5 Chat Threads & History

The side panel keeps a history of your chat threads. You can start new conversations, switch between threads, and revisit past chats. Threads are stored locally on your device.

3.6 Side Panel Themes

The side panel has its own theme setting, separate from the Quick Button themes. Switch between light and dark modes and other visual styles in the side panel settings.

5Settings & Advanced Features

To open Settings: click the Courier Web logo inside the toolbar that appears when you highlight text, or right-click and choose Settings.

5.1 Destination Settings

Choose your default destination for all prompts. Set per-site routing rules — for example, always sending YouTube highlights to ChatGPT and Gmail highlights to Claude.

Per-site rules format: hostname=destination (one per line, e.g. youtube.com=chatgpt)

5.2 Prompt Queue (Advanced)

The prompt queue lets you batch up multiple sends and run them all at once, rather than opening a new tab for each. Most useful with ChatGPT. Enable under Advanced → Prompt Queue in Settings.

5.3 History

Courier Web keeps a local history of everything you've sent. Free users get a smaller history cap; Pro users get a larger one. You can search, review, and reuse past entries from the History page.

🔒
Your history is stored locally and never sent to any server. To open: Settings → Open History.

5.4 Backup & Restore

Export all your settings — templates, quick buttons, destinations, and preferences — as a backup file, and import them on another computer or after reinstalling.

Backup typeWhat's included
Standard backupSettings, templates, quick buttons, destinations, preferences. No API keys.
Full encrypted backupEverything above plus API keys and Pro token, encrypted with a passphrase you set.
⚠️
After reinstalling: use the Restore button (not Activate) — it will re-verify your license key and restore your Pro status automatically.

5.5 Languages

Courier Web is localized into English, German, Spanish, French, Japanese, and Brazilian Portuguese. The extension automatically uses your browser's language setting.

7API Keys

API keys let the Side Panel send prompts directly to an AI provider using your own account. All keys are stored locally in your browser — Courier Web never sees or transmits them.

7.1 Where to Enter Your Keys

2
Scroll to API KeysScroll down until you reach the API Keys section.
3

4
Key is confirmedA message appears showing the key has been saved, with a masked preview (e.g. sk-an...k29f).

Select the modelOpen the Side Panel, click the model dropdown, and choose a model from that provider.

7.3 Clearing or Updating a Key

To replace a key, paste the new one into the same field and click Save — it overwrites the old one. To remove a key entirely, click the Clear button next to the field. The Side Panel stops using that provider immediately.

7.4 API Keys vs Destinations

You don't need an API key at all to use the Highlight & Prompt feature. API keys are only needed for the Side Panel.

8Paste Mode

When you send highlighted text to a destination, Courier Web needs to get your content into that site's input box. There are three methods — called paste modes — that control exactly how this happens.

8.1 The Three Paste Modes

ModeWhat it doesBest for
Auto-injectOpens the destination site and automatically types your text directly into the chat or compose box — no manual action needed.ChatGPT (most reliable)
Paste helperOpens the destination site and a small helper tab. Text is copied to your clipboard automatically. Switch to the destination tab and press Cmd/Ctrl+V.Claude, Gemini, DeepSeek, Groq, most AI sites
Open onlyOpens the destination site but does not copy or inject anything. You navigate and paste manually.Gmail, X, Google — where URLs already carry the content, or when you want full control

8.2 Default Paste Modes Per Destination

DestinationDefault modeWhy
ChatGPTAuto-injectChatGPT's input box is well-supported for direct injection
ClaudePaste helperUses a complex contenteditable editor; helper is more reliable
GeminiPaste helperUses a custom Angular input that can block auto-injection
DeepSeekPaste helperEditor requires manual paste for consistent results
GroqPaste helperPaste helper ensures reliable delivery
GrokPaste helperPaste helper ensures reliable delivery
PerplexityPaste helperPaste helper ensures reliable delivery
KimiPaste helperPaste helper ensures reliable delivery
OpenRouterPaste helperPaste helper ensures reliable delivery
GmailOpen onlyGmail compose URLs already include subject and body text
X (Twitter)Open onlyShort text is passed directly in the tweet URL
GoogleOpen onlySearch query is passed directly in the URL
FacebookPaste helperRequires browser permission grant first

8.3 How to Change Paste Mode

  1. Open Settings (click the Courier Web icon or the logo in the highlight toolbar).
  2. Scroll to the Destination section.
  3. Expand Paste mode — you'll see a grid with every destination listed.
  4. Use the dropdown next to each destination to switch modes.
  5. Changes save automatically.

Click Reset paste modes to restore all destinations to recommended defaults.

8.4 Problems Paste Mode Can Fix

SymptomTry this
Text never appears in ChatGPTSwitch ChatGPT to Paste helper
Text gets cut off or garbled in ChatGPTSwitch to Paste helper — clipboard has no size limit
Claude / Gemini opens but nothing is pastedEnsure mode is Paste helper, then press Cmd/Ctrl+V
Facebook: clicking button does nothingGrant the Facebook permission when prompted, or switch to Paste helper
Text appears twice in the inputSwitch to Open only or Paste helper
Helper tab opens but closes before you can pasteText is already on your clipboard — switch tabs and press Cmd/Ctrl+V
Gmail / X opens but text isn't in compose boxSwitch from Open only to Paste helper
💡
Quick rule of thumb: Auto-inject stops working after a site update → switch to Paste helper. Paste helper feels like too many steps → try Auto-inject. Just want to open the site yourself → use Open only.

9Troubleshooting

ProblemFix
Toolbar doesn't appear on highlightsRefresh the page. Courier Web may not activate on pages that were open before installation. Check that the minimum character length in Settings isn't set too high.
Menu not showing after right-clickClose the right-click menu and right-click again. Chrome menus don't update while they're open.
Clicking a button does nothingOpen DevTools (F12 → Console). A "blocked by client" message means a Chrome permission is missing. Go to chrome://extensions → Courier Web → Details → Site access.
"Address already in use" (Ollama)Not an error — Ollama is already running in the background. No action needed.
Ollama model not found / 404Run ollama list and copy the exact model name including any version tag (e.g. llama3.2:latest). The Model ID must match exactly.
CORS error with OllamaRestart Ollama with: OLLAMA_ORIGINS="*" ollama serve
403 error from Hugging FaceYour API token may be missing or the model is gated. Accept the terms of use on the model's page before trying again.
YouTube transcript not availableThe video may not have captions. Check if the CC button appears in the YouTube player.
Templates not appearing in menuClose and re-open the right-click menu. Templates update when saved but the change doesn't show until the menu is re-opened.
Extension changes not taking effectGo to chrome://extensions and click the reload icon on the Courier Web card, then refresh the page.
Pro not showing after activationClick Restore in the Pro section of Settings. If you recently reinstalled, use Restore (not Activate) to re-verify your saved license key.
🩺
Run Diagnostics
Settings → Advanced → Diagnostics / Health runs a full check of your extension state, permissions, and endpoint connectivity. Use this first when something unexpectedly stops working.
